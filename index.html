ViInVaRa

Video Interval Variable Rater

This software can be used for experiments wherein a variable has to be rated on a scale while watching a video. The results are presented as an interactive graph and can be downloaded as a CSV table or JPG.
This work is based on MacIntyre's Idiodynamic Software/ the Anion Variable Tester.
